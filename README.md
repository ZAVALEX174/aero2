# aero2
Детальный анализ кода редактора технических чертежей

1. ОБЩЕЕ ОПИСАНИЕ СИСТЕМЫ
Это веб-приложение для создания технических чертежей с использованием библиотеки Fabric.js. Программа позволяет:

Основные возможности:
Рисование линий с параметрами (L, I, K, W)

Добавление изображений (оборудование, датчики, вентиляторы и т.д.)

Разделение линий в точках пересечения

Редактирование свойств объектов

Сохранение/загрузка чертежей

Отмена/повтор действий

Работа с сеткой
